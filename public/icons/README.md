# PWA Icons

Les icônes suivantes doivent être créées pour la PWA :

- `icon-192.png` : 192x192 pixels
- `icon-512.png` : 512x512 pixels  
- `apple-touch-icon.png` : 180x180 pixels (pour iOS)

**Design recommandé :**
- Fond : #6C63FF (violet ShiftPilot)
- Forme : Carré arrondi (border-radius ~20%)
- Logo : Symbole de planning/calendrier en blanc
- Style : Moderne, épuré, reconnaissable

**Outils pour créer les icons :**
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- Figma/Photoshop avec export PNG

Pour l'instant, utilisez un placeholder ou générez-les depuis le SVG de référence.
